# 🚀 Job Tracker App

A full-stack job tracking web application built to help users manage and monitor job applications efficiently.

This project demonstrates my full-stack development skills using **React, Node.js, Express, and SQLite**.

---

## 🌟 Overview

The **Job Tracker App** helps users organize and track job applications in one place.

Users can:

- Add job applications
- Track application progress
- Update application status
- Delete job applications
- View all job applications in a clean dashboard

This project focuses on real-world **full-stack development concepts**, including:

- CRUD operations
- API integration
- Database management
- Frontend and backend communication

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Jj879304/job-tracker-app.git
```

### 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

### 3️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```
