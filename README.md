# Job Tracker App

A full-stack web application that helps users track job applications.

## Features

- Add jobs
- View all jobs
- Change application status
- Delete jobs
- SQLite database storage

## Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Node.js
- Express.js
- SQLite
